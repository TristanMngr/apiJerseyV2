# apiJerseyV2
Project: apiJerseyV2

Version: 1.0.1

How to run using maven?

1. Clonar esta branch

git clone https://github.com/ccantero/apiJerseyV2.git

2. Compilar

mvn clean install

3. Correr utilizando jetty.

mvn jetty:run

Eg: http://localhost:8080/
